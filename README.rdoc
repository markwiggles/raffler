== Ember raffler

This is the Railscast project built using Emberjs frontend and Rails backend.

Note:

- Coffeescript used in the video has been converted to plain Javascript
- Deprecated code updated as at January 15, 2014 and versions below

<tt>Versions used:</tt>
* Ember 1.9.1
* Ember Data 1.0.0-beta.14.1
* Handlebars 2.0.0
* jQuery 1.11.2

<tt> See video and script at http://railscasts.com</tt>
